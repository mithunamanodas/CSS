CSS position

The CSS position property is used to specify where an element is displayed on a webpage.01 It determines the final location of the element when paired with the top, right, bottom, and left CSS properties.0 A fixed element does not leave a gap in the page where it would normally have been located. The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute, or sticky). The position property is used to create engaging websites by positioning elements like text and images.
